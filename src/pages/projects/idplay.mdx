## The Tech Stack

This application was built using:

- **Laravel** for Framework building the application system
- **Mysql** for static database system
- **Bootstrap** for rapid styling and responsiveness
- **Postman** for Cheking end point API
- **Jquery** for makes it easier to manipulate the DOM

## Features

- 💻 **Customer Data Management** to showcase projects
- 📱 **Responsive Design** to work across various screen sizes
- 🖥️ **Service Package Management** for customizable viewing experiences
- ⚡ **Fast Performance** powered by Vite for quick load times
- 👥 **Customer Support Provides a ticketing system** for easy extensibility
- 🔐 **User Authentication** (Sign Up & Sign In)

## Implementation Highlights

- Built a retro terminal-style UI using styled-components for a clean design.
- Implemented Vite for fast and optimized development and build processes.
- Used TypeScript for type safety and improved maintainability.
- Designed the portfolio to be interactive with clear instructions for users to navigate.
